Git 常用命令

发表于 2018-10-07 更新于 2021-07-20 分类于 Git 阅读次数：
本文字数： 1.5k 阅读时长 ≈ 1 分钟

记录 Git 日常操作常用命令

git config

Git级别：system(系统所有用户) < global(当前用户) < local(当前仓库)

  • 查看配置信息

  • 新增或修改

  • 删除用户配置信息

git init

  1. 把已有项目代码纳入 Git 管理
  2. 新建项目直接使用 Git 管理

git clone

  • clone
  • clone 指定分支
  • clone 指定tag
  • clone 指定commit

git commit

-a 指定标签名，-m 指定说明文字

git branch

  • 创建分支
  • 切换分支
  • 删除分支
  • 重命名分支
  • 查看分支

git tag

  • 新增 tag
  • 查看 tag
  • 删除 tag
  • 推送 tag 到远程

git mv

  • 重命名文件

git log

  • 查看仓库 commit 历史日志

git help

更多命令

git other

  • 查看当前项目远程仓库地址
  • 修改仓库地址
