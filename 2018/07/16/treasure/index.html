必备的一些辅助插件，让你效率翻倍

通用

  1. Alibaba Java Coding Guidelines：阿里巴巴 Java 代码规范
  2. GrepConsole：控制台日志自定义
  3. GsonFormat：json 生成对应的实体 bean
  4. GenerateAllSetter：对象中所有属性生成 set 方法
  5. ignore：文件忽略
  6. Lombok：简化实体 bean
  7. Markdown Navigator：Markdown 文件支持
  8. PlantUML integration：UML 文件支持
  9. Translation：语言翻译

Android

  1. Android ButterKnife Zelezny：ButterKnife 相关支持
  2. Android Material Design Icon Generator：Material icon 生成
  3. Android Parcelable code generator：Parcelable 代码生成
  4. Android Postfix Completion：.toast，.log 等快捷写法
  5. Android Resource Usage Count：资源使用统计
  6. Android Studio Prettify：模板代码生成，比如：findViewById
  7. ADB Idea：ADB 相关命令
  8. EventBus3 Intellij Plugin：EventBus 相关支持
  9. LayoutFormatter：格式化 XML 布局
  10. SQLScout：SQL 相关支持

Java

  1. Alibaba Cloud Toolkit：服务部署工具
  2. JsonToKotlinClass：json 生成 kotlin 对象
  3. MyBatisCodeHelperPro：mybatis 支持及 XML 代码调试
